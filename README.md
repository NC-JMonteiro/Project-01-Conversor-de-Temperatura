# 🌡️ Conversor de Temperatura

Este é um projeto simples desenvolvido em **C#** que realiza a conversão de temperaturas de **Celsius para Fahrenheit** através de uma aplicação de **console**.

O projeto foi criado como parte da disciplina **Algoritmos e Lógica de Programação**, com o objetivo de praticar conceitos básicos de programação.

---

## 📌 Sobre o projeto

O programa solicita ao usuário uma temperatura em **graus Celsius**, aplica a fórmula de conversão e exibe o resultado equivalente em **Fahrenheit**.

A interação acontece diretamente pelo **terminal/console**.

---

## 🧮 Fórmula utilizada

A conversão de Celsius para Fahrenheit utiliza a seguinte fórmula: F = (C × 9/5) + 32

Onde:

- **C** = Temperatura em Celsius  
- **F** = Temperatura em Fahrenheit  

---

## 💻 Exemplo de execução

Digite a temperatura atual em Celsius: 25

Celsius: 25C°
Fahrenheit: 77F°
